# CommonTrack

**CommonTrack** was created to assist students applying to colleges by providing detailed, easy-to-understand information about each college on the Common App. Key details include ranking, size, type, deadlines, test policy, and recommendations.

## Features
* **Home Page:** Explains purpose and ranking criteria.
* **My College List:** Track up to 20 colleges and add deadlines to calendars.
* **Search All Colleges:** Browse and search the full Common App college list.
* **Ranked Colleges:** View QS-ranked colleges with detailed scores.
* **About Section:** Share purpose, motivation, and inspiration behind the app.

## Technology & Frameworks
* **Python (Flask):** Backend development, routing, and core logic.
* **CS50 SQL:** Simplified database operations.
* **Werkzeug.security:** Secure password hashing.
* **Functools:** Session management and login redirection.
* **Google Calendar API:** Sync and manage college deadlines.

## Motivation
The idea for CommonTrack was inspired by personal challenges experienced while applying to colleges. By streamlining the process and offering an organized way to manage deadlines and critical information, the app aims to simplify and enhance the college application experience for students.
